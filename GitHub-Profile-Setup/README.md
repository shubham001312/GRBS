## 🚀 Getting Started

### Local Development

1. Clone the repository:
   ```bash
   git clone https://github.com/shubham001312/Shubham-Mallick.git
   ```

2. Open `index.html` in your browser — no build step needed!

### Deploy to GitHub Pages

From the `portfolio-site/` directory, run the deploy script:
```bash
cd portfolio-site
bash deploy.sh
```

> ⚠️ The deploy script copies files to the `Shubham-Mallick-deploy` repo and pushes to GitHub Pages.
