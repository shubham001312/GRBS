## 🎯 What is GRBS?

**GRBS (GPT Roadmap By Shubham)** is a comprehensive, interactive learning tracker built for AI/ML engineers. It features a structured 15-phase curriculum, real-time progress tracking, and a beautiful mobile-first interface — all running entirely in your browser with zero backend.

## ✨ Features

- 📚 **15 Progressive Phases** — From CS foundations to building GPT from scratch
- 🎯 **Interactive Dashboard** — Overall completion ring, readiness meters, study streaks
- 🗺️ **Full Roadmap** — Collapsible phase cards with topics, resources, and milestones
- 📊 **Visual Analytics** — Radar charts, bar charts, activity heatmap
- 🎯 **Career Roadmaps** — 5 paths: Internship → Placement → AI Engineer → LLM Specialist → GPT Builder
- 🔍 **Global Search** — Instantly find any topic, project, or resource
- 💾 **Offline Support** — PWA with service worker caching
- 📱 **Mobile-First** — Designed for 375px+ screens
- 👤 **Personalized** — Username onboarding with localStorage persistence
- 📤 **Export/Import** — Backup and restore your progress as JSON

## 🚀 Quick Start

```bash
# Clone the repository
git clone https://github.com/shubham001312/GRBS.git
cd GRBS

# Open in browser (no build step needed!)
# Windows
start index.html
# macOS
open index.html
# Linux
xdg-open index.html
```

## 📁 Project Structure

```
GRBS/
├── index.html              # Main app entry point (PWA-ready)
├── manifest.json           # PWA manifest for installability
├── sw.js                   # Service worker for offline caching
├── css/
│   └── styles.css          # Mobile-first stylesheet (warm cream/red theme)
├── js/
│   ├── data.js             # All 15 phases with topics & resources
│   ├── state.js            # localStorage persistence, streak, activity tracking
│   ├── app.js              # App init, navigation, search, tab switching
│   ├── dashboard.js        # Dashboard: rings, readiness meters, greeting
│   ├── roadmap.js          # Roadmap: collapsible phase cards, topics, resources
│   ├── projects.js         # Projects: filterable card grid with status
│   ├── progress.js         # Progress: radar/bar charts, activity heatmap
│   ├── goals.js            # Goals: 5 career path timelines
│   └── about.js            # About: developer card, tech stack, mission
└── assets/
    ├── icon-192.png        # PWA icon (192x192)
    └── icon-512.png        # PWA icon (512x512)
```
