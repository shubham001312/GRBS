## 🎯 What is Wifto?

**Wifto** is a gamified application environment featuring key-specific usage tracking and search capabilities. Built as a web-based project exploring gamification concepts and user engagement patterns.

## ✨ Features

- 🎮 **Gamified Interface** — Engaging user experience
- 🔍 **Search Capabilities** — Key-specific content discovery
- 📊 **Usage Tracking** — Monitor app interactions
- 📱 **Responsive** — Works across devices

## 🚀 Quick Start

```bash
git clone https://github.com/shubham001312/Wifto.git
cd Wifto
start index.html
```
