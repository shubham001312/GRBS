## 🎯 What is MINIMAL-CLOCK?

**MINIMAL-CLOCK** is a beautifully designed minimal clock application featuring a timer, user data monitoring, data manipulation, visual representation, DND (Do Not Disturb) mode, and ADO (Auto Display Off). A clean, aesthetic prototype of a modern clock app.

## ✨ Features

- ⏰ **Minimal Clock UI** — Clean, aesthetic time display
- ⏱️ **Timer** — Built-in countdown timer
- 📊 **Data Monitoring** — Track and display user data
- 🎨 **Visual Representation** — Beautiful data visualization
- 🔕 **DND Mode** — Do Not Disturb functionality
- 💤 **ADO** — Auto Display Off for battery saving
- 📱 **Responsive** — Works on all screen sizes

## 🚀 Quick Start

```bash
# Clone the repository
git clone https://github.com/shubham001312/MINIMAL-CLOCK-VERSION-1.0.0.git

# Open in browser
start index.html
```
