## 🎯 What is ROSHNI?

**ROSHNI** is a beautifully designed photo gallery web application featuring a responsive grid layout, smooth transitions, and an immersive viewing experience. Built with clean HTML, CSS, and JavaScript.

## ✨ Features

- 🖼️ **Photo Gallery** — Beautiful grid-based image layout
- 📱 **Responsive** — Adapts to all screen sizes
- 🎨 **Smooth Transitions** — Polished animations and hover effects
- 🔍 **Image Viewer** — Click to view full-size images
- ⚡ **Fast Loading** — Optimized image delivery
- 🌐 **Deployed** — Live on GitHub Pages

## 🚀 Quick Start

```bash
# Clone the repository
git clone https://github.com/shubham001312/ROSHNI.git
cd ROSHNI

# Open in browser
start index.html
```
