## 🎯 What is GUZU?

**GUZU** is a privacy-focused Android browser application built with Flutter. It features multiple search engine support, a clean responsive UI, and is designed for secure, private browsing on mobile devices.

## ✨ Features

- 🔒 **Private Browsing** — Focus on privacy and security
- 🔍 **Multi-Engine Search** — Choose from multiple search engines
- 📱 **Cross-Platform** — Built with Flutter for Android, iOS, Web, Linux, macOS, Windows
- 🎨 **Clean UI** — Modern, responsive interface
- ⚡ **Optimized Performance** — Smooth app flow and navigation
- 🏗️ **Well-Structured** — Clean architecture for maintainability

## 🚀 Quick Start

### Prerequisites
- Flutter SDK
- Android Studio or VS Code

### Run the App

```bash
# Clone the repository
git clone https://github.com/shubham001312/guzu--ANDROID_SOURCE_CODE.git
cd guzu--ANDROID_SOURCE_CODE

# Get dependencies
flutter pub get

# Run on connected device/emulator
flutter run
```
