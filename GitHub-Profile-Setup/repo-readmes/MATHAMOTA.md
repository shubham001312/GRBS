## 🎯 What is MATHAMOTA?

**MATHAMOTA** is an interactive mathematics visualization tool built with JavaScript. It explores mathematical concepts through visual representations and interactive simulations.

## ✨ Features

- 📐 **Math Visualization** — Interactive mathematical graphics
- 🎨 **Canvas Rendering** — Smooth visual output
- 📱 **Responsive** — Works on all devices
- ⚡ **Lightweight** — Pure JavaScript, no dependencies

## 🚀 Quick Start

```bash
git clone https://github.com/shubham001312/MATHAMOTA.git
cd MATHAMOTA
start index.html
```
