## 🎯 What is Goru?

**Goru** is an AI-powered application built with modern TypeScript, Vite, and Google's Gemini API. Designed as a fast, responsive web experience with AI capabilities.

## ✨ Features

- 🤖 **AI-Powered** — Built with Google Gemini
- ⚡ **Fast** — Vite-powered development and build
- 📱 **Responsive** — Works across all devices
- 🔒 **Secure** — Local API key management
- 🎨 **Modern UI** — Clean, intuitive interface

## 🚀 Quick Start

```bash
# Clone the repository
git clone https://github.com/shubham001312/Goru.git
cd Goru

# Install dependencies
npm install

# Set your Gemini API key
echo "GEMINI_API_KEY=your-key-here" > .env.local

# Start development server
npm run dev
```
