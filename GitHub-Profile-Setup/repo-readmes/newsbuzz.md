## 🎯 What is NewsBuzz?

**NewsBuzz** is a modern news aggregation web application that fetches and displays news from multiple sources with a clean, responsive interface. Built as a full-stack web project with a backend API and frontend client.

## ✨ Features

- 📱 **Responsive Design** — Works on all screen sizes
- 🔄 **Real-time Updates** — Fresh news delivered to your browser
- 🔍 **Category Filtering** — Browse news by topic
- 🎨 **Clean UI** — Modern card-based layout
- ⚡ **Fast Loading** — Optimized for performance
- 🌐 **Deployed** — Live on GitHub Pages

## 🚀 Quick Start

```bash
# Clone the repository
git clone https://github.com/shubham001312/newsbuzz.git
cd newsbuzz

# Open in browser
start index.html
```
