## 🎯 What is Wifto Project?

**Wifto Project** is a gamified application environment featuring key-specific usage tracking and search capabilities. A web-based project exploring gamification and user engagement.

## ✨ Features

- 🎮 **Gamified Interface** — Engaging user experience
- 🔍 **Search Capabilities** — Key-specific content discovery
- 📊 **Usage Tracking** — Monitor app interactions
- 📱 **Responsive** — Works across devices
- 🌐 **Deployed** — Live on GitHub Pages

## 🚀 Quick Start

```bash
git clone https://github.com/shubham001312/wifto-project.git
cd wifto-project
start index.html
```
