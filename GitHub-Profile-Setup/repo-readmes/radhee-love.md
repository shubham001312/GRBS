## 🎯 What is Radhee Love?

**Radhee Love** is an interactive, animated web experience featuring beautiful visuals, audio integration, and smooth CSS animations. A creative project showcasing frontend design skills.

## ✨ Features

- 💕 **Thematic Design** — Beautiful love-themed visual experience
- 🎵 **Audio Integration** — Multiple audio tracks (happy, love, sad)
- ✨ **CSS Animations** — Smooth, polished transitions
- 📱 **Responsive** — Works on all devices
- 🌐 **Deployed** — Live on GitHub Pages

## 🚀 Quick Start

```bash
git clone https://github.com/shubham001312/radhee-love.git
cd radhee-love
start index.html
```
