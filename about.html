About This Project

GPT Roadmap By Shubham (GRBS) is a free, open learning portal that guides B.Tech students from programming basics all the way to building and deploying Large Language Models. Every resource linked is free. Every project is practical. Every phase builds on the last.

Built with: HTML · CSS · JavaScript · ❤️

No frameworks. No databases. Just a student helping students.
